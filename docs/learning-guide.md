## 用語集（随時追加）

| 用語 | 一言説明 |
|------|---------|
| RDT | Recurrent-Depth Transformer。同じブロックをT回繰り返して「思考」する |
| RoPE | Rotary Position Embedding。位置情報を回転で埋め込む手法 |
| GQA | Grouped Query Attention。KVヘッドを減らしてキャッシュを節約 |
| MLA | Multi-Latent Attention。DeepSeek方式のKV圧縮 |
| MoE | Mixture of Experts。トークンごとに異なる専門家FFNを使う |
| ACT | Adaptive Computation Time。問題の難しさに応じてループ回数を自動調整 |
| LoRA | Low-Rank Adaptation。少ないパラメータで効率的に学習/適応 |
| SSE | Server-Sent Events。サーバーからクライアントへの一方向ストリーム |
| RAG | Retrieval-Augmented Generation。外部DB検索結果をLLMに与えて回答精度を上げる |
| LLMO | LLM Optimization。AIサーチ向けコンテンツのスコアリング |
| SBOM | Software Bill of Materials。依存ライブラリの脆弱性リスト |
